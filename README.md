🏅 Paris Olympics 2024 – Power BI Data Analysis Dashboard
📌 Project Overview

This project presents an interactive Power BI dashboard analyzing Paris Olympics 2024 data. The dashboard provides insights into athlete participation, medal distribution, country performance, and historical Olympic trends through dynamic and user-friendly visualizations.

📊 Key Features

Analysis of 11,000+ athletes across 200+ countries

KPI tracking: Total Medals, Gold/Silver/Bronze distribution, Gender-wise participation

Country rankings and top-performing nations analysis

Historical medal trend analysis across Olympic years

Interactive slicers (Year, Country, Gender) and drill-through functionality

Map-based visualizations for global performance insights
